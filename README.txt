Completing the hardware and coding exercizes for the Elegoo Electronics Fun Kit using the Arduino UNOR4 WiFi, breadboards, the Arduino IDE, and various electronic components. 

Done: 

1. Install the IDE, add libraries and Open serial monitor
2. Program internal LED to blink
3. Provide power to LED via 5V output and breadboard, various resistors
4. RGB LED with PWM analog output
5. Use buttons to control LED
6. Control active buzzer
7. Control passive buzzer and play a melody
8. Control 8 LEDs with 3 pins using 74HC595
